Regles cles
Present Perfect vs Preterit
Present Perfect = action passee avec impact present. Preterit = moment precis termine.
We have signed the deal (resultat actuel) / We signed it last Tuesday (moment precis)
Faux amis a bannir
Ces mots grillent immediatement un non-natif.
Eventually = Finalement (pas eventuellement) / Actual = Reel (pas actuel) / Sensible = Raisonnable (pas sensible)
Suisse alemanique
"Gruezi" est le salut formel suisse, pas "Hallo". A utiliser avec les profs et inconnus.
Gruezi = Formel Suisse / Hallo = Informel / Danke schoen = Merci
